## 二、斯诺克核心规则梳理

### 2.1 球配置
| 球类型 | 数量 | 分值 | 击打顺序 |
|--------|------|------|----------|
| 红球 | 15颗 | 1分 | 先击打 |
| 黄球 | 1颗 | 2分 | 红球清台后 |
| 绿球 | 1颗 | 3分 | 红球清台后 |
| 棕球 | 1颗 | 4分 | 红球清台后 |
| 蓝球 | 1颗 | 5分 | 红球清台后 |
| 粉球 | 1颗 | 6分 | 红球清台后 |
| 黑球 | 1颗 | 7分 | 红球清台后 |

### 2.2 完整击打流程（已实现）

#### 2.2.1 流程概览
```
┌─────────────────────────────────────────────────────────────────┐
│ 阶段1: 红球阶段（15套红+彩）                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 红球 → 彩球（复位）→ 红球 → 彩球（复位）→ ...         │   │
│  │ 共15套，彩球打进后复位，可重复击打                      │   │
│  └─────────────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────────────┤
│ 阶段2: 过渡阶段（最后一颗红球后的彩球）                      │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 红球（第15颗）→ 任意彩球（复位）→ 进入彩球阶段        │   │
│  │ 这颗彩球也复位，之后开始按顺序击打彩球                  │   │
│  └─────────────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────────────┤
│ 阶段3: 彩球阶段（按顺序击打）                               │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 黄球 → 绿球 → 棕球 → 蓝球 → 粉球 → 黑球             │   │
│  │ 按分值从小到大顺序击打，不再复位                      │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
```

#### 2.2.2 详细规则表
| 阶段 | 条件 | 可击打目标 | 彩球是否复位 |
|------|------|------------|--------------|
| 红球阶段 | 有红球 + 上一次未击打红球 | 红球 | - |
| 红球阶段 | 有红球 + 上一次击打红球 | 任意彩球 | 是 |
| 过渡阶段 | 红球=0 + 上一次击打红球 | 任意彩球 | 是 |
| 彩球阶段 | 红球=0 + 已击打过渡彩球 | 按顺序：黄→绿→棕→蓝→粉→黑 | 否 |

#### 2.2.3 禁止操作表（已改为按钮禁用）
| 场景 | 是否允许 | 处理方式 |
|------|----------|----------|
| 有红球时先击打彩球（未打红球） | ❌ | 彩球按钮禁用 |
| 击打红球后连续再击打红球 | ❌ | 红球按钮禁用 |
| 彩球阶段击打错误顺序的彩球 | ❌ | 未轮到的彩球按钮禁用 |
| 所有球打完后继续击打 | ❌ | 所有按钮禁用 |

#### 2.2.4 单杆得分规则
- 连续进球累加单杆分数
- 犯规或失误则重置单杆分数
- 切换选手重置单杆分数
- 记录每位选手的最高单杆分数

#### 2.2.5 剩余最高分计算
**公式**：剩余最高分 = 剩余红球数 × 8 + 27（彩球总分）

**说明**：
- 每个红球(1分) + 黑球(7分) = 8分/套
- 彩球总分：黄2 + 绿3 + 棕4 + 蓝5 + 粉6 + 黑7 = 27分

#### 2.2.6 超分规则
**超分条件**：当前选手领先分数 > 剩余最高分

**示例**：
- 台面剩5个红球+所有彩球，剩余最高分为 5×8+27=67分
- 领先68分以上即为超分
- 只剩彩球时，剩余最高分为27分，领先28分以上即为超分

### 2.3 犯规罚分规则
| 犯规类型 | 罚分 | 说明 |
|----------|------|------|
| miss（未击中） | 4分 | 未击中任何球 |
| cue_ball_fall（母球落袋） | 4分 | 母球掉入袋中 |
| wrong_ball（打错球） | 4分 | 未按顺序击打 |
| no_contact（未接触） | 4分 | 未接触目标球 |
| foul_red（红球犯规） | 4分 | 击打红球时犯规 |
| foul_yellow（黄球犯规） | 2分 | 击打黄球时犯规 |
| foul_green（绿球犯规） | 3分 | 击打绿球时犯规 |
| foul_brown（棕球犯规） | 4分 | 击打棕球时犯规 |
| foul_blue（蓝球犯规） | 5分 | 击打蓝球时犯规 |
| foul_pink（粉球犯规） | 6分 | 击打粉球时犯规 |
| foul_black（黑球犯规） | 7分 | 击打黑球时犯规 |

### 2.4 犯规罚分弹窗（新增）
点击犯规按钮后弹出罚分选择弹窗，提供4、5、6、7四个分值选项：

| 分值 | 用途 | 说明 |
|------|------|------|
| 4分 | 普通犯规 | 击空、误击红球等 |
| 5分 | 蓝球犯规 | 误击蓝球 |
| 6分 | 粉球犯规 | 误击粉球 |
| 7分 | 黑球犯规 | 误击黑球 |

**交互流程**：
1. 点击犯规按钮 → 弹出居中弹窗
2. 选择罚分 → 执行罚分并切换选手
3. 点击弹窗外部 → 关闭弹窗（取消操作）

## 八、比赛流程说明

### 8.1 抢局赛模式
1. 配置页面选择抢局赛模式，设置总局数（如5局）
2. 系统自动计算胜利条件（如5局3胜）
3. 每局结束后显示结算弹窗，选择"开始下一局"或"结算比赛"
4. 当某选手获胜局数达到胜利条件时，自动结束比赛并显示最终结果

### 8.2 计时赛模式
1. 配置页面选择计时赛模式，设置比赛时长
2. 比赛在规定时间内进行
3. 时间结束时自动结算，分数高者获胜

### 8.3 超分处理
1. 当当前选手领先分数超过台面剩余最高分（超分）时，显示"超分"提示
2. 超分后仍可继续击打，但比赛结果已基本确定
3. 可随时点击"结算本局"或"结束比赛"提前结束
